# Proyecto de acceso remoto a VNA

## Instalación

### Clonar repositorio

```bash
git clone https://github.com/fPecc/me2_vna_instvirtual.git
```

## Iniciar aplicación

### Server

Abrir una consola en la carpeta donde se clonó el proyecto e insertar los siguientes comandos:

```bash
cd backend
flask run
```

### UI

```bash
cd ui
node app.js
```
